Sistema de Gestión Inmobiliaria y CRM

Desarrollado en Odoo ERP | En Producción (Sistema Interno)

Desarrollo e implementación de un módulo personalizado, sobre el ecosistema ERP de Odoo. Este sistema interno centraliza y optimiza la gestión integral de propiedades, propietarios y el flujo de ventas/alquileres, integrándose de forma nativa con el módulo de CRM.

Funcionalidades Clave

  • Gestión de Propiedades: Registro completo (área, linderos, ubicación, disponibilidad).
  • Manejo Multimoneda: Conversión dinámica USD/Local en tiempo real para precios y comisiones.
  • Integración CRM: Conexión directa entre propiedades y pipeline de ventas (Leads automáticos).
  • Gestión de Propietarios: Historial y sincronización inteligente con oportunidades en curso.
  • UI/UX Personalizada: Módulos para vistas fluidas con componentes y diálogos personalizados.

Tecnologías

Odoo ERP Python XML / JavaScript PostgreSQL
